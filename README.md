# Movie App

A React application for browsing and saving favorite movies using The Movie Database (TMDB) API.

## Features

- Browse popular movies
- Search movies by title
- Save favorites (persisted to localStorage)
- Responsive movie grid layout

## Tech Stack

- React 19
- React Router DOM
- Vite
- TMDB API

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run linter
npm run lint
```

## Project Structure

```
src/
├── components/     # Reusable UI components (NavBar, MovieCard)
├── pages/          # Route pages (Home, Favorites)
├── contexts/       # React Context providers
├── services/       # API integration
└── css/            # Component styles
```
